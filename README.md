# My Portfolio

This repository contains the codebase for my personal portfolio website, designed and developed to showcase my projects, skills, and experiences.

## Project Overview

The portfolio website is a modern, responsive, and interactive platform that highlights my work, background, and professional achievements. It includes sections for:

- **About Me**: A brief introduction and background information.
- **Projects**: A collection of past and current projects with descriptions and links.
- **Skills**: A breakdown of technical and soft skills.
- **Experience**: Details of relevant work experience and education.
- **Contact**: A way for visitors to reach out via email or social media.

## Technologies Used

- **Frontend**: React.js, HTML, CSS, JavaScript
- **Styling**: Tailwind CSS for modern and flexible styling
- **Animations**: Framer Motion for smooth UI animations
- **Deployment**: Hosted using Vercel or Netlify for fast and secure delivery

## Getting Started

To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sidp24/my-portfolio.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd my-portfolio
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```

5. **Open the website**:
   - Visit `http://localhost:3000/` in your browser.

## Deployment

The portfolio can be deployed using Vercel or Netlify:

- **Vercel**:
  ```bash
  vercel deploy
  ```
- **Netlify**:
  ```bash
  netlify deploy
  ```
